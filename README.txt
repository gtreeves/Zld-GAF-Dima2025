A MATLAB-based code for Zld-sfGFP and GAF-sfGFP image analysis and fitting models to data for the paper:

Dima, S. S. & Reeves, G. T. (2025). Bulk-level maps of pioneer factor binding dynamics during Drosophila maternal-to-zygotic transition Development, dev.204460. DOI: 10.1242/dev.204460

To run the RICS time course image analysis, see the "RICS_timecourse" folder. To generate the figures from the paper, see the "Generate figures" folder.
